
# Project Design Document (PDD)
## School Website Revamp with GenAI Integration
### Sri Laxmi Narayan Saraswati Vidya Mandir

---

## 1. Project Overview

**Project Name:** SLNSVM Website Revamp & AI Enablement  
**Client:** Sri Laxmi Narayan Saraswati Vidya Mandir  
**Location:** Bhagwanpur, Vaishali, Bihar  
**Board:** CBSE  

The project aims to redesign and rebuild the existing school website into a modern, secure, student‑centric digital platform with integrated Generative AI capabilities (LangGraph + AI Chatbot) to improve academic support, administration efficiency, and stakeholder communication.

---

## 2. Objectives

- Provide a modern, mobile‑first digital presence
- Improve student academic engagement
- Automate administrative operations
- Enable AI‑powered learning assistance
- Ensure CBSE compliance
- Centralize communication for parents, teachers, and students
- Provide scalability for future expansion

## 8. Technical Architecture

|-------|------------|
| Frontend | React / Next.js |
| Backend | Node.js / Django |
| Database | PostgreSQL |
| AI Engine | OpenAI / Claude + LangGraph |
| Vector DB | Pinecone / Weaviate |

| Payments | Razorpay |
