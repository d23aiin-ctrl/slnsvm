
# Request for Proposal (RFP)
## School Website Revamp with GenAI – SLNSVM

## 1. Project Background
Sri Laxmi Narayan Saraswati Vidya Mandir (CBSE) intends to revamp its existing website into a modern, AI-enabled digital platform.

## 2. Project Objectives
- Student-centric learning platform
- Parent & teacher portals
- Admin automation
- GenAI chatbot & LangGraph agents
- CBSE compliance

## 3. Scope of Work
- UI/UX design
- Frontend & backend development
- Database setup
- AI system integration
- Deployment
- Training & documentation

## 4. Deliverables
- Fully functional website
- Admin dashboard
- AI chatbot
- LangGraph workflows
- Source code
- Documentation

## 5. Vendor Requirements
- Minimum 3 years experience
- Education domain projects
- AI integration experience
- Security best practices

## 6. Proposal Format
- Company profile
- Technical approach
- Timeline
- Team structure
- Cost breakdown
- Maintenance plan

## 7. Evaluation Criteria
- Technical expertise
- AI capability
- Cost-effectiveness
- Past experience
- Support quality

## 8. Project Timeline
Estimated 4–5 months

## 9. Budget
To be proposed by vendor (module-wise)
